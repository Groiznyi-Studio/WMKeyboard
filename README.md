# WMKeyboard
WMKeyboard - Turns the keyboard into a musical instrument

# How to play
1)Run application

2)Press q, a, z, w, s, x, e, d, c, r, f, v, t, g, b, y, h, n, u, j, m -- to play sound

3)Press num1, num2, num3, num4, num5, num6 -- to set theme

4)Press num-, num+ -- to set time play sound

5)Enjoy!!!

# Warning
1)TimePlay must be at least 50 and over 10000!!!

2)If you try to go against the rules, a message will appear that this cannot be done!!!
